### 나의 취향을 기록하고 분석하는 취향 아카이브 서비스

## 💥 Boom! – 요즘 나의 '붐'은?

<img src="readme_img/메인화면.png" alt="메인 화면" width="100%">

: Boom! 은 사용자가 요즘 빠져 있는 것을 카드로 기록하고,
카테고리 기반 점수(PAEF)에 따라 취향을 분석해 나만의 성향을 정리하고,
유사한 취향의 유저, 카드, 키워드를 발견하도록 도와주는 서비스입니다.

<br>

### 프로젝트 설명

이 프로젝트는 데이터베이스설계(CSC4020-01) 과목의 프로젝트입니다. 

- **Backend**: Java 17, Spring Boot 3.5.7, MySQL  
- **Frontend**: React, Tailwind CSS

### 주요 기능

#### 💥 1. 사용자의 취향을 담은 카드 생성

- 사용자가 좋아하는 콘텐츠/취미/장소 등을 카드 형태로 기록할 수 있어요. 이 카드에 저장된 데이터를 바탕으로 사용자의 취향을 분석해요.

<img src="readme_img/카드생성1.png" alt="이미지" width="100%">
<img src="readme_img/카드생성2.png" alt="이미지" width="100%">


#### 💥 2. PAEF 점수 기반의 취향 분석 리포트 제공

- Productivity / Activity / Emotional / Focus 총 4가지 점수를 기반으로 사용자의 취향을 정량화해요. Boom Level을 가중치로 사용해서 사용자의 PAEF 점수를 계산하고, 당신의 페르소나 유형을 제공합니다. 

- My Boom 페이지에서 취향 분석 리포트와 최다 선택 카테고리 TOP5, 나의 취향 키워드 TOP5, 내가 생성한 카드들, 수집한 카드들을 조회할 수 있어요. 

<img src="readme_img/마이페이지.png" alt="이미지" width="100%">

- 또한, 상대방의 프로필에서 나와의 취향 유사도가 얼마나 되는지도 알 수 있어요!

<img src="readme_img/사용자프로필.png" alt="이미지" width="100%">


#### 💥 3. 인기 / 최신순 카드 조회 + 카드 추천

- 최근에 생성된 카드, 내가 좋아할 만한 카드, 인기 카드를 조회할 수 있어요. 

- 마음에 드는 카드를 수집함에 저장할 수 있어요. 수집한 카드들은 My 페이지에서 모아볼 수 있습니다.

- 만약 추천하고 싶은 카드를 발견했다면, BoomUP을 해보세요! 많은 BoomUP을 받을 수록 인기 카드가 될 확률이 높아져요. 


<img src="readme_img/카드목록.png" alt="이미지" width="100%">


#### 💥 4. 생성한 카드를 이미지로 저장

- 생성한 카드는 웹에서도 볼 수 있지만, 이미지로 다운로드해서 앨범에 저장할 수 있어요. 나의 취향 카드를 차곡차곡 모으는 재미가 있어요. 

<img src="readme_img/카드저장.png" alt="이미지" width="100%">

### ✨ Boom! 서비스가 특별한 이유


1) **취향을 기록하고 분석할 수 있어요!**
- 카드를 기반으로 사용자의 감정·활동·집중·생산성 성향을 점수화하여 성향을 분석하고 진단하여 16개의 페르소나 유형으로 제공합니다! 


2) **나와 비슷한 취향인 사람을 찾을 수 있어요!**
- 다른 사용자와의 취향 유사도를 확인할 수 있어요. PAEF 점수를 기반으로 타 사용자와의 유사도를 계산하고 


3) **내가 좋아할 만한 다른 사용자의 취향 카드를 추천해줘요!**
- PAEF 성향 점수를 이용해서 유클리드 거리 기반으로 취향 카드를 추천 받을 수 있어요. 


4) **카드를 이미지로 저장해서 수집할 수 있어요!**
- 단순히 게시물로 업로드하고 끝나는 게 아니라, 이미지로 저장해서 외부에 공유할 수 있어요. 이것이야말로 진정한 아카이브가 아닐까요? 나만의 취향 카드를 모아보세요!
